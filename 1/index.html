Colorizing the Prokudin-Gorskii Photo Collection

Adnan Aman

Project Background

In the early 20th century, photographer Sergei Prokudin-Gorskii captured monochromatic glass plate images using separate red, green, and blue filters. These glass plates represent an important part of photographic history, and using image processing techniques, we can combine these plates into a full-color image.

Approach

The process involves splitting the original images into three parts (Blue, Green, Red) and aligning the color channels using normalized cross-correlation (NCC). To optimize for larger images, an image pyramid approach was used, where the image is scaled down progressively to find the optimal alignment, and then scaled up to refine it.
